# Robot Error LED Guide

|Color | LED1 (Module) | LED2 (Level) | LED3 (Info)|
|------|---------------|--------------|------------|
|Green | FPGA          | Error        | Soccer Connect Fail |
|Blue  | Kicker        | Warning      | WiFi Connect Fail |
|White | IMU           | Info         | |
|Red   | Radio         | Fatal (very bad) |Boot Fail |
