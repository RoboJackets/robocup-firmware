Detailed Description

template<typename DIGITAL_IN, size_t NUM_PINS = 4>
class RotarySelector< DIGITAL_IN, NUM_PINS >

This class facilitates reading the value from a rotary selector.

This class is templated so that it generalizes I/O functionality to not only IO expander pins but also standard mTrain I/O pins.

Example usage: RotarySelector<DigitalIn> selector({p5, p6, p7, p8}); printf("selector value: %d\r\n", selector.read());
