constexpr auto rtp::ControlMessage::VELOCITY_SCALE_FACTOR = 1000
static

body{X,Y,W} are multiplied by this value before being sent over the radio and must be then divided by this value on the receiving side.

This is to avoid loss of precision when sending float velocity values across the air as ints.
